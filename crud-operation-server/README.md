# CRUD Operation - Server (Express + MongoDB)

This is the backend part of the CRUD application.

## Technologies
- Node.js
- Express.js
- MongoDB (Mongoose)

## How to Run
```bash
cd crud-operation-server
npm install
node index.js
Features
Create User (POST)

Get Users (GET)

Update User (PUT)

Delete User (DELETE)

MongoDB
Connected using Mongoose to your own MongoDB Atlas or local MongoDB.
